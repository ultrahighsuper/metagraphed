# Changelog

All notable changes to `@jsonbored/metagraphed` (the typed TypeScript client) are
documented here. From the next release onward this file is generated by
release-please from Conventional Commits.

## 0.2.0

- Typed TypeScript client for the metagraphed API, generated from the OpenAPI
  contract. Adds the typed health-trends endpoint over the `0.1.0` bootstrap.
- Published to npm via OIDC trusted publishing (provenance, no long-lived token).

## 0.1.0

- Initial bootstrap release of the generated typed client.

# Changelog

All notable changes to the `metagraphed` Python client are documented here. From
the next release onward this file is generated by release-please from Conventional
Commits.

## 0.1.1

- Add a descriptive `User-Agent`, surface the API error envelope in raised
  `MetagraphedError`s, and guard JSON parsing of error responses (over `0.1.0`).
- Apache-2.0 licensed; published to PyPI via OIDC trusted publishing.

## 0.1.0

- Initial release of the typed Python client for the metagraphed API.
